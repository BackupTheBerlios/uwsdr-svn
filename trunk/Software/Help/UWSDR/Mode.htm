This sets the SDR mode (actually the DSP code, the SDR knows nothing about the mode), to one of:

  • CW Narrow
  • CW Wide
  • USB
  • LSB
  • AM
  • FM Narrow
  • FM Wide

The setting of the mode can also affect the AGC, tuning speed and the filter bandwidth, depending on the values set in the Modes Preferences and the Step Size Preferences. The filter bandwidth can be overridden by the Filter Selection control below the Mode Selection.
