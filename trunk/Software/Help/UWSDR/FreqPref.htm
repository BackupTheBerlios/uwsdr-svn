This screen sets the transmit and receive limits of the SDR. The sdr description file sets the absolute hardware limits whereas this screen is used to set the regulatory limits. The receive limits must be the same or within the hardware limits, and the transmit limits must be the same or within the receive limits. These values are in Megahertz.

Shift 1 is used by the SHIFT 1 button, this is typically used for repeater operation. This value is given in Kilohertz.

Shift 2 is used by the SHIFT 2 button, this is typically used for repeater operation. This value is given in Kilohertz.

The Offset is a means by which a difference in frequency between the display on the GUI and reality may be removed. This is particularly important when the SDR is not locked to a high quality frequency standard. This value is given in Hertz.

The transmit limits and the Frequency Shift on this screen are not operational when the SDR is receive only. The Frequency Offset is disabled unless the SDR is an actual UWSDR. If the frequency is not correct for a fixed frequency SDR like a SoftRock, then this should be adjusted by editing the associated .sdr file.
