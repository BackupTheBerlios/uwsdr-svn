The UWSDR GUI may be started in a number of ways. Depending on the operating system and the options selected in GUI Setup not all of them may be available to you.

  • Clicking on icons created by GUI Setup in either a program menu or from the Desktop.


  • Typing the command in at a command line shell, in the form

    UWSDR <name>

    where <name> is the name given in GUI Setup. This option does not work correctly with Mac OS X.


  • Typing the command in at a command line shell, in the form

    UWSDR

    in this case the last GUI configuration used will be taken. This option does not work correctly with Mac OS X.

If no configuration name is given, or there was no previous configuarion used, or the name passed is invalid, then an error box will appear and the GUI will not start.
